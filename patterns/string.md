# String Pattern

## 🧠 Core Idea
Strings are arrays of characters → apply array techniques.

---

## 🚀 When to Use
- Pattern matching
- Parsing
- Substrings
- Palindromes

---

## 🧩 Common Techniques
- Two pointers
- Sliding window
- HashMap

---

## 🧠 Intuition
Treat string like array:
- Traverse
- Compare
- Modify (carefully)

---

## ⚠️ Common Mistakes
- Ignoring edge cases (empty string)
- Case sensitivity
- Index out of bounds
- String immutability (in some languages)

---

## 🔄 Time & Space Complexity
Varies by problem

---

## 🧩 Example Problems
- Longest Common Prefix
- Valid Palindrome
- Roman to Integer

---

## 🧠 Key Insight
Strings are about:
➡️ **character-level manipulation**
