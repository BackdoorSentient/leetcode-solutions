# Math Pattern

## 🧠 Core Idea
Use mathematical properties instead of brute force.

---

## 🚀 When to Use
- Numbers
- Digits manipulation
- Divisibility
- Patterns

---

## 🧩 Common Techniques
- Modulo (%)
- Division
- Logarithms
- Bit manipulation (advanced)

---

## 🧠 Intuition
Math problems often have:
➡️ Hidden patterns or shortcuts

---

## ⚠️ Common Mistakes
- Overflow (in other languages)
- Using strings unnecessarily
- Missing edge cases (negative numbers, zero)

---

## 🔄 Time & Space Complexity
Usually O(1) or O(log n)

---

## 🧩 Example Problems
- Palindrome Number
- Reverse Integer

---

## 🧠 Key Insight
Math reduces:
➡️ **computation using formulas**
