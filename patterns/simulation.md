# Simulation Pattern

## 🧠 Core Idea
Directly follow problem instructions step by step.

---

## 🚀 When to Use
- Movement problems
- Grid/coordinates
- Step-by-step processes

---

## 🧠 Intuition
No optimization needed  
Just implement carefully

---

## ⚡ Approach
- Initialize state
- Process each step
- Update state

---

## ⚠️ Common Mistakes
- Missing conditions
- Wrong updates
- Not tracking state properly

---

## 🔄 Time & Space Complexity
- Time: O(n)
- Space: O(1)

---

## 🧩 Example Problems
- Robot Return to Origin

---

## 🧠 Key Insight
Simulation is about:
➡️ **accuracy over optimization**
