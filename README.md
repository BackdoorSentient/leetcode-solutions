## Pattern Decision Guide

```text
Subarray / substring problem?
├── Fixed size window              -> Sliding Window
├── Variable size window           -> Sliding Window
├── Count/sum of subarrays         -> Prefix Sum + HashMap
└── Exactly K type counting        -> AtMost(K) trick or Prefix Sum

Sorted array?
├── Find target/index              -> Binary Search
├── Pair/triplet                   -> Two Pointers
└── Rotated sorted array           -> Modified Binary Search

Need previous value quickly?
├── Pair sum / frequency           -> HashMap
└── Prefix seen before             -> Prefix Sum + HashMap

Linked list?
├── Cycle                          -> Fast & Slow Pointers
├── Middle                         -> Fast & Slow Pointers
└── Reverse / merge                -> Pointer handling

Choices / combinations?
└── Backtracking

Best answer from many choices?
└── Dynamic Programming / Greedy
```
