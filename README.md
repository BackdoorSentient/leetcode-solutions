# 🧠 PATTERN DECISION TREE (VERY IMPORTANT)

```
Is it array/string?

├── Subarray / Substring?
│   ├── Fixed size → Sliding Window
│   ├── Variable size → Sliding Window
│   └── Sum related → Prefix Sum

├── Pair / Triplet?
│   ├── Sorted → Two Pointers
│   └── Unsorted → HashMap

├── Need optimal answer in range?
│   └── Binary Search

├── Maximum / Minimum subarray?
│   └── Kadane

├── Overlapping intervals?
│   └── Intervals / Greedy

---

Is it Tree?

├── Traversal?
│   ├── DFS → Recursion
│   └── BFS → Queue

├── Path / Depth?
│   └── DFS

---

Is it Graph?

├── Connected components?
│   └── DFS / BFS / Union Find

├── Dependency ordering?
│   └── Topological Sort

---

Is it choices / combinations?

└── Backtracking

---

Is it optimization over choices?

└── Dynamic Programming
```
